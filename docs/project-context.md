## Project Summary

NeuroNest is an intelligent assistive sensory-regulation device for children with ASD and ADHD. It is designed to help children manage sensory overload, calm emotions, and improve focus during lessons.

The research document describes NeuroNest as a portable multi-sensory device using:

- Soft blue or green LED light.
- Gentle rhythmic vibration.
- Tactile surfaces for touch stimulation.
- Optional physiological stress monitoring, such as pulse or skin response.
- Optional feedback summaries for teachers or parents, with consent.

## Prototype Direction

The recommended prototype is a browser-based interactive 3D model, not a Blender-first project.

Reason:

- The user has no Blender experience.
- The deadline is very short.
- A browser prototype is easier to build, present, and interact with.
- The prototype only needs to demonstrate the concept clearly, not prove realistic manufacturing.

## Intended Deliverable

One interactive 3D browser prototype showing a rounded handheld NeuroNest device.

Expected prototype features:

- Rotatable and zoomable 3D product model.
- Rounded child-safe casing.
- Soft glowing LED zone.
- Tactile texture zones.
- Animated vibration zones.
- Sensor pad for stress monitoring concept.
- Clickable feature hotspots.
- Mode controls: Calm, Focus, Alert.
- Small side panel explaining the selected feature and simulated stress response.

## Presentation Goal

The prototype should help the presenter explain:

1. NeuroNest is a portable sensory-regulation device for children with ASD and ADHD.
2. The LED zone provides calming customizable light.
3. The vibration zone provides gentle haptic feedback.
4. Tactile zones support self-regulation through touch.
5. The sensor pad represents stress monitoring.
6. NeuroNest can discreetly activate calming support and provide adult feedback with consent.

## Constraints

- One-evening build.
- No backend required.
- No real sensor data.
- No medical claims beyond a concept demonstration.
- Keep the interface simple enough for a first-time presenter.
- Do not depend on learning Blender UI unless the assignment later requires a `.blend` file.
